# <small>:nut_and_bolt:</small> GetOcrText


   
### Formats

This action can accept and produce the following content type formats.

| Accepts | Produces |
|-----|-----|
|<ul><li>`application/pdf` _(.pdf)_</li></ul>|<ul><li>`text/plain` _(.txt)_</li></ul>|

### Usage

```js
{
  "kind": "PdfGetOcrText",
}
```
