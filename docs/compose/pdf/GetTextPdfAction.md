# <small>:nut_and_bolt:</small> GetText

Extracts text from PDF documents.

### Accepts

  - `application/pdf`

### Outputs

  - `text/plain`

### Usage

```js
{
  "kind": "PdfGetText",
  "followLayout": false
}
```
#### Properties

**`followLayout`**  `boolean`

Attempt to follow the layout of the document when extracting text.
